FadeOut


Inherits from: Object

Purpose


Add release to any synthdef to avoid clipping. 


Provide a curved envelope to simplify the adding of a fade-in and fade-out envelope to a SynthDef, thereby preventing clips at start and end of the synth. 


Usage


//:Using FadeOut:


a = { Out.ar(0, WhiteNoise.ar(0.1 * FadeOut.kr)) }.play;


//:---

a.set(\fadeout, 10, \gate, 0);


//:By comparison, release without use of FadeOut:

// This uses the fadeout code that is part of Function:play


a = { WhiteNoise.ar(0.1) }.play;


//:---

a.release(10);


//: ----
