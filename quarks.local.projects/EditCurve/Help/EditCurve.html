Summary of classes. 


Curve Holds and manages the data for one curve

CurveFile Stores and loads data for curves

CurveGUI Graphic interface for curves

Interpolation Interpolation between points of curves

ParaCurveGUI Handles the settings for the gui display (-> CurveGuiOptions)

PtS Extension of Point, adds several parameters used in Curve

PtSExt Experimental: Implement different data extrapolation algorithms

for projection of curves to time and other parameters. 

May be substituted by separate classes implementing those algotithms.

See file FonctionsSpecialesCurves. 

SelectorCurve Selects part of a curve to work with on display etc.

Titre GUI utility class for displaying the parameters on CurveGUI


File: FonctionsSpecialesCurves: Implement different data extrapolation algorithms

(see also: PtSExt)
