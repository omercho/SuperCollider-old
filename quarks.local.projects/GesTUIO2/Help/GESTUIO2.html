TouchOSC



Usage



TouchOSC.new // create and activate a new session



TouchOSC.deactivate // deactivate the default TouchOSC instance



To customize the behavior of TouchOSC, store functions to the following instance variables: 


sessionCreated

sessionChanged

sessionEnded


Like this: 


TouchOSC.new.sessionCreatedAction = { | session |

session.id.postln;

{ SinOsc.ar(400 + 1000.rand, 0, EnvGen.kr(Env.perc(level: 0.1), doneAction: 2)) }.play;

};




TouchOSC.new.sessionCreatedAction = { | session |

// session.id.postln;

{ WhiteNoise.ar(EnvGen.kr(Env.perc(level: 0.1), doneAction: 2)) }.play;

};
